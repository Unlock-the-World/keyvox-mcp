# Contributor Covenant 行動規範

## 私たちの誓約

メンバー、コントリビューター、リーダーである私たちは、年齢、体格、目に見える / 見えない障害、民族性、性自認 / 性表現、経験レベル、教育、社会経済的地位、国籍、容姿、人種、宗教、性的指向の如何に関わらず、ハラスメントのないコミュニティを目指して活動することを誓約します。

私たちは、オープンで歓迎的、多様で包括的、健全なコミュニティに貢献するよう行動・対話することを誓約します。

## 私たちの基準

ポジティブな環境作りに貢献する行動の例:

- 他者への共感と思いやりを示すこと
- 異なる意見・視点・経験を尊重すること
- 建設的なフィードバックを与え、また受け入れること
- 自分の過ちに対して責任を持ち、影響を受けた人々に謝罪し、その経験から学ぶこと
- 個人だけでなくコミュニティ全体にとって最善のことを優先すること

容認されない行動の例:

- 性的な言葉や表現を使うこと、性的な誘いを行うこと
- 侮辱的な発言、差別的な発言、個人または政治的な攻撃
- 公的または私的なハラスメント
- 明示的な許可なく他者の住所、電子メールアドレスなどの個人情報を公開すること
- 物理デバイス制御 (スマートロック解錠等) を悪用しようとする示唆 / 助言
- その他、職業上不適切と合理的に判断される行動

## 実施責任

コミュニティリーダーは、容認されない行動に対して、適切で公正な是正措置を取る責任があります。

リーダーは、行動規範に沿わないコメント、コミット、コード、Wiki 編集、Issue、その他の貢献を削除、編集、または拒否する権限を持ち、適切な場合は理由を説明します。

## 適用範囲

この行動規範は、コミュニティの全ての場、ならびに個人がコミュニティを公式に代表している場で適用されます。コミュニティの代表とは、例えば公式メールアドレスの使用、公式ソーシャルメディアアカウントでの投稿、オンライン / オフラインイベントでの代理人としての行動などです。

## 適用

虐待的、嫌がらせ、その他容認されない行動の事例は、コミュニティリーダーへ以下の窓口で報告できます:

- **KEYVOX サポートポータル**: <https://support.keyvox.co> (チケット件名に `[Code of Conduct]` を付記)

すべての報告は速やかに、公正に確認・調査されます。

コミュニティリーダーは、報告者のプライバシーと安全を尊重する義務があります。
